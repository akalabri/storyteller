## gcloud in PATH (Windows)
- **New terminals:** User PATH already includes the Google Cloud SDK; open a new terminal and run `gcloud`.
- **Current terminal:** From repo root run:
  ```powershell
  . .\add_gcloud_path.ps1
  ```
  Then you can run `gcloud` in that session.
